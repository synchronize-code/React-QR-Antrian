# 😃 QRAN

QRAN (QR Antrian) With React JS Hooks & Apollo Client

## 🧰 Feature
 - Register Queue
 - Dashboard User
 - Dashboard Operator
 - Board
 - Data With Graphql Subscription
## 🚀 Getting started

### Installation 

```javascript 
npm install 
```

### Run React

```javascript 
npm start 
```
